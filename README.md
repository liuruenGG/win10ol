# Windows 10 26H2 Concept (Windows on Web)

一个基于 Web 技术栈构建的 Windows 10 26H2 概念版操作系统仿真项目。本项目旨在探索前端交互的极致可能性，还原流畅、现代的桌面操作体验。

## 下载方式
将这个项目包 git clone 到你要存储的位置中，然后从项目包启动 CMD 以执行之后的操作。

## 快速启动

```bash
# 安装依赖
npm install

# 启动项目
npm run dev
```
>注：运行时请全程不要关闭 CMD!!!
